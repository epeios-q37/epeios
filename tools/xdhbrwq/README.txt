xdhbrwq is a CGI program based on the Epeios framework which renders XDHTML module in web browsers
xdhbrwq is developed by Claude SIMON (http://q37.info/contact/).

See following files in the 'doc' directory :
- 'COPYING' for the license,
- 'CHANGELOG' for the changelog of the program.

There is another 'CHANGELOG' file in 'src/epeios', which concerns the Epeios framework.
As this programm is based on the Epeios framework, some entries may also concern this program.

More comprehensive documentation may be found in the 'doc' directory at the root of the package.

xdhbrwq home page :  http://q37.info/computing/epeios/tools/xdhbrwq/.
