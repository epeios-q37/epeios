# xppq-php

## Presentation

The *Epeios* XML [preprocessor](http://q37.info/s/t/xppq/) and parser as [*Zend*](http://q37.info/s/x/Zend_engine/) (*PHP*) extension.

For more information, see http://q37.info/tools/xppq/.
